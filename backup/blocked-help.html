Main Site Blocked?

Follow these steps to deploy your own backup site on GitHub Pages.

1. Open the GitHub Repo

Go to: github.com/immcrab/games

2. Fork the Repository

Click Fork to make your copy.

3. Go to Settings

Open the Settings tab in your fork.

4. Go to “Pages”

Scroll down and click Pages.

5. Deploy the Site

  • Deploy from a branch
  • Branch: main
  • Folder: /root

Click Save.
